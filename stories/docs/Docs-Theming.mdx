# Theming

AstronautUI uses CSS variables to enable instant theming (dark/light) and brand customization.

## Tokens

`src/styles/tokens.css` defines semantic tokens:

- **Colors:** `--aui-color-surface`, `--aui-color-overlay`, `--aui-color-text`, `--aui-color-primary`, etc.
- **Radii:** `--aui-radius-sm/md/lg/xl`
- **Spacing:** `--aui-space-1..6`
- **Shadows, z-index, motion**
